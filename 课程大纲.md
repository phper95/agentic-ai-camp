# Agentic AI 智能体开发行动营详细课程大纲

## 课程核心信息

- 课程周期：5 周实战特训
- 学习形式：12 + 小时视频精讲 + 5 小时直播干货分享 + 项目实战带练 + 2 套作业练习 + 群内答疑
- 核心技术栈：MCP、LangGraph、LangFuse、LlamaFactory、vLLM、LoRA、Docker、FastAPI、Streamlit 等
- 实战成果：6 个企业级实战项目 + 可商用代码与部署方案

## 第 1 周：掌握工具调用，让智能体接入真实世界

### 本周目标

1. 掌握 Function Calling 的原理和实现方法
2. 理解 MCP 协议，实现标准化工具集成
3. 掌握提示词工程和上下文管理技巧
4. 不使用框架手动构建一个简单的 Agent

### 课程内容

1. 任务 01：工具调用 - 用 Function Call 实现文件搜索工具
2. 任务 02：掌握 MCP 协议 - 标准化接入天气查询工具
3. 任务 03：MCP 客户端开发 - 集成 DeepSeek 和 LangChain 框架
4. 任务 04：深入理解智能体 - 手搓一个 Agentic AI 的最小框架
5. 任务 05（直播）：提示词与上下文工程 - 让智能体精准理解复杂任务需求

## 第 2 周：构建文献研究助手（多智能体协同系统），实现复杂任务自动化

### 本周目标

1. 掌握 LangGraph 的核心概念和开发方法
2. 理解状态管理、记忆持久化、路由决策
3. 掌握人机协同和并行协作的设计模式
4. 完成深度研究助手项目部署和测试

### 课程内容

1. 任务 06：从链式到图状态 - 构建一个简单的 LangGraph 应用
2. 任务 07：智能决策 - 构建智能路由，实现记忆管理
3. 任务 08：人机协同与并行协作 - 构建可控、可扩展的智能体系统
4. 任务 09：DeepResearch 深度研究助手的创建与配置
5. 任务 10：DeepResearch 深度研究助手的容器化部署与测试
6. 任务 11（直播）：LangGraph 高级技巧与作业点评

## 第 3 周：“旅小智” AI 旅行规划智能体设计（多角色智能体），实现生产环境部署

### 本周目标

基于 FastAPI + Streamlit + LangGraph + MCP 技术栈，完成多角色旅行规划智能体设计与生产环境部署

### 课程内容

1. 任务 12：架构设计 - AI 旅行规划智能体系统架构设计
2. 任务 13：全栈开发 - 用 Streamlit + FastAPI 完成前后端开发与系统集成
3. 任务 14：Docker 容器化实战 - 一键部署智能体系统
4. 任务 15（直播）：生产环境部署实战与问题排查

## 第 4 周：掌握 Langfuse 技术，建立智能体评估与安全监控体系

### 本周目标

1. 建立智能体评估体系和指标设计方法
2. 掌握 Langfuse 追踪和评估技术
3. 构建 LLM 安全监控系统
4. 实现生产环境的持续监控和优化

### 课程内容

1. 任务 16：建立追踪评估体系 - 用 Langfuse 追踪智能体执行过程
2. 任务 17：Langfuse 集成实战 - 追踪 OpenAI、LangChain 和 LangGraph 应用
3. 任务 18：使用 Langfuse 实现智能体在线 / 离线评估
4. 任务 19：安全监控实战 - 构建 LLM 攻击检测系统
5. 任务 20（直播）：生产环境持续监控与性能优化实战

## 第 5 周：微调垂直领域（医疗）模型，打造专业 Agentic AI

### 本周目标

1. 理解模型微调的适用场景和价值
2. 掌握 LoRA/PEFT 高效微调技术
3. 使用 LlamaFactory 进行模型微调
4. 掌握微调数据集构建和效果评估
5. 完成垂直领域模型的微调和部署

### 课程内容

1. 任务 21：模型微调入门 - 用 LoRA 低成本优化垂直领域模型
2. 任务 22：数据集构建实战 - 用 Easy Dataset 快速准备微调数据
3. 任务 23：LlamaFactory 微调实战 - 微调 Qwen/DeepSeek 模型
4. 任务 24：模型评估与优化 - 科学评估微调效果
5. 任务 25（直播）：模型部署与智能体集成实战
